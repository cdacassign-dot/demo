Q1. Create and Initialize Git Repository




Step 1 — Check Git Installation
First, verify Git is installed on your Ubuntu system:
bash
git --version
If not installed, run:
bash
sudo apt update
sudo apt install git -y




git config --global user.name "Utkarsh Kumar"
git config --global user.email "your_email@example.com"


Create a new repository named git_lab on your local system.
Initialize it using Git.
Create a file info.txt and add your name and date.
Check the repository status.
Add the file to staging and commit with message "Initial commit" .


mkdir git_lab
cd git_lab
git init
echo "Name: Utkarsh Kumar" > info.txt
echo "Date: $(date)" >> info.txt
git status
git add info.txt
git commit -m "Initial commit"




Q2. Connect to GitHub and Push Code
Create a repository named git_lab_remote on GitHub.
Link your local repository ( git_lab ) to the remote repository.
Push your committed file to the remote repository.
Verify the file is uploaded on GitHub.


cd ~/git_lab
git remote add origin https://github.com/your-username/git_lab_remote.git
git remote -v
git branch
git push -u origin master




Q3. Clone Repository and Make Changes
Clone the repository git_lab_remote to a new directory.
Create a new file data.txt and add some sample content.
Add and commit the file with message "Added data file" .
Push the changes back to GitHub.




cd ~
git clone https://github.com/your-username/git_lab_remote.git
cd git_lab_remote
echo "This is sample data." > data.txt
echo "Created by: Utkarsh Kumar" >> data.txt
echo "Purpose: Git Lab Assignment Q3" >> data.txt
git status
git add data.txt
git commit -m "Added data file"
git push origin master




Q4. Branching and Merging
Create a new branch named feature .
Switch to the feature branch.
Modify info.txt (add new line: “Feature branch update”).
Commit the changes.
Switch back to main branch.
Merge the feature branch into main .
Delete the feature branch after merge.




cd ~/git_lab_remote
git branch feature
git checkout feature
echo "Feature branch update" >> info.txt
git add info.txt
git commit -m "Added feature branch update"
git checkout master
git merge feature
git branch -d feature
git push origin master
